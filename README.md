### 설치 및 실행 방법

1. 프로젝트를 클론합니다.

```bash
git clone https://github.com/KSH-KOR/KSH-KOR.github.io.git
```

2. 프로젝트 폴더로 이동합니다.

```bash
cd KSH-KOR.github.io
```

3. 필요한 패키지를 설치합니다.

```bash
npm install
```

4. 개발 서버를 실행합니다.

```bash
npm start
```
